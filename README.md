# Premier-league-srs-model
Editted on basis of lcbeas (https://github.com/lcbeas/nba_srs_ratings) 

How to use:
- For Windows - 
1. clone the website in cmd (please pip install `git` beforehand if you haven't installed it)
2. open the downloaded folder in your ide (vscode, pycharm, etc.)
3. run `pip install -q -r requirements.txt` in terminal
4. run `python -m scripts.generate_site_data` in terminal
5. run `python -m http.server 8000 --directory site` in terminal
6. NOW YOU SEE IT! ^^
